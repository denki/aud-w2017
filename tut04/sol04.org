** Teilaufgabe c

Seien ρ(S) = ρ(A) = { (ba)ⁿb ∣ n ∈ ℕ }.  Zu zeigen: ⟦{ba}A⟧(ρ) = ρ(S).

Beweis:

⟦{ba}A⟧(ρ)
= ⟦{ba}⟧(ρ) ⋅ ρ(A)
= (⟦ba⟧(ρ))* ⋅ ρ(A)
= {ba}* ⋅ ρ(A)
= {ba}* ⋅ { (ba)ⁿb | n ∈ ℕ }
= { (ba)ᵏ | k ∈ ℕ } ⋅ { (ba)ⁿb | n ∈ ℕ }
= { (ba)ᵏ(ba)ⁿb | k, n ∈ ℕ }
= { (ba)ⁿb | n ∈ ℕ }
= ρ(S)
